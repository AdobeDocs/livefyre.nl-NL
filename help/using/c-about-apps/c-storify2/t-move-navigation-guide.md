# Navigatiegids verplaatsen{#move-the-navigation-guide}

1. Schakel in de Designer Storify 2 App de **[!UICONTROL Navigation Guide]** schakeloptie in.
1. Selecteer boven, links of rechts in de **[!UICONTROL Orientation]** vervolgkeuzelijst.
